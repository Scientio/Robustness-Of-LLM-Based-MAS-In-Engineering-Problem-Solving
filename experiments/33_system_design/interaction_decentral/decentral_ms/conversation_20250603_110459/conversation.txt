AGENT 1: To calculate the pressure loss, we can use the Darcy-Weisbach equation: 

ΔP = f * (L/D) * (ρ * v² / 2)

Given that we assume the friction factor f is always 25/Re, we first need to calculate the Reynolds number (Re) for water flow.

Re = (ρ * v * D) / μ

For water, let's assume ρ = 1000 kg/m³ and μ = 0.001 Pa·s. Plugging in the values:

Re = (1000 * 0.01 * 0.1) / 0.001 = 1000

Now, we can find f:

f = 25/Re = 25/1000 = 0.025

Now substituting into the Darcy-Weisbach equation:

ΔP = 0.025 * (10/0.1) * (1000 * (0.01)² / 2) = 0.025 * 100 * (1000 * 0.0001 / 2) = 0.025 * 100 * 5 = 12.5 Pa

So, the pressure loss in the pipe is 12.5 Pa.
